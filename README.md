# Strona przedmiotu Systemy Operacyjne na wydziale MINI PW

Strona używa generatora Hugo (testowane wersje 0.152).

Żeby zbudować stronę do katalogu `public/` wystarczy odpalić `hugo` bez argumentów. 

Z użyciem `hugo server` można uruchomić serwer dostarczający stronę.

# MINI PW  Operating Systems course website

The website utilizes Hugo (tested version 0.152).

Run `hugo` without params to build the website to the `public/` directory.

Run `hugo server` to start local server hosting the website.
